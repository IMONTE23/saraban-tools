# Saraban Tools

เครื่องมือช่วยปิดงานอัตโนมัติสำหรับระบบสารบรรณอิเล็กทรอนิกส์ (Electronic Document Management System) ช่วยลดขั้นตอนการทำงานที่ซ้ำซ้อนและประหยัดเวลาในการจัดการเอกสารปริมาณมาก

## 🌟 คุณสมบัติเด่น (Features)

*   **ทำงานอัตโนมัติ (Automation):** ช่วยกดปิดงานตามรายการในตารางโดยอัตโนมัติ
*   **รองรับ 2 โหมดการทำงาน:**
    *   **โหมดนาฬิกา (ระหว่างดำเนินการ):** สำหรับกดปุ่มสัญลักษณ์นาฬิกาในแถวตาราง
    *   **โหมดแก้ไข (รอลงทะเบียน):** สำหรับกดปุ่มแก้ไขในเมนูจัดการของแถวตาราง
*   **ปรับแต่งได้ยืดหยุ่น:**
    *   กำหนดจำนวนแถวที่ต้องการดำเนินการได้ (หรือเลือกทำทั้งหมด)
    *   ตั้งค่าดีเลย์ (Delay) ระหว่างแถวได้เพื่อความเสถียรและความปลอดภัยของระบบ
*   **ระบบบันทึกประวัติ (Real-time Log & Progress):** แสดงสถานะการทำงาน จำนวนแถวที่ทำสำเร็จ และข้อผิดพลาดในขณะปฏิบัติงานแบบเรียลไทม์
*   **เป็นมิตรต่อความเป็นส่วนตัว:** ทำงานอยู่บนเบราว์เซอร์ของผู้ใช้โดยตรง ไม่มีการส่งข้อมูลใด ๆ ออกไปยังภายนอก

---

## 🛠 วิธีติดตั้ง (Installation)

1.  ดาวน์โหลดหรือโคลน (Clone) ซอร์สโค้ดของโปรเจกต์นี้ลงบนเครื่องคอมพิวเตอร์ของคุณ
2.  เปิดเบราว์เซอร์ Google Chrome หรือเบราว์เซอร์ที่รองรับ Chromium
3.  พิมพ์ `chrome://extensions/` ที่ช่องกรอก URL แล้วกด Enter
4.  เปิดใช้งาน **"โหมดนักพัฒนาซอฟต์แวร์" (Developer mode)** ที่มุมขวาบนของหน้าจอ
5.  คลิกปุ่ม **"โหลดส่วนขยายที่คลายซิปแล้ว" (Load unpacked)** ที่มุมซ้ายบน
6.  เลือกโฟลเดอร์ของโปรเจกต์นี้ที่บันทึกไว้ในเครื่อง

---

## 🚀 วิธีใช้งาน (Usage Instructions)

1.  เข้าสู่ระบบสารบรรณอิเล็กทรอนิกส์ และไปยังหน้าตารางรายการเอกสารที่ต้องการปิดงาน
2.  คลิกที่ไอคอน **Saraban Tools** บนแถบเครื่องมือของเบราว์เซอร์
3.  ตั้งค่าตัวเลือกต่าง ๆ ตามต้องการ:
    *   **โหมดการเริ่มทำงาน:** เลือกโหมดระหว่าง "ระหว่างดำเนินการ (ปุ่มนาฬิกา)" หรือ "รอลงทะเบียน (ปุ่มแก้ไข)"
    *   **จำนวนแถว:** กรอกจำนวนแถวที่ต้องการให้ระบบทำ หรือเลือก "ทำทั้งหมดในหน้านี้"
    *   **ดีเลย์การทำงาน (ms):** กำหนดเวลาหน่วงระหว่างเปลี่ยนแถว (แนะนำ 800ms ขึ้นไป เพื่อลดภาระของเซิร์ฟเวอร์ระบบสารบรรณ)
4.  คลิกปุ่ม **เริ่มทำงานอัตโนมัติ**
5.  ระบบจะดำเนินการทีละแถวจนเสร็จสิ้น คุณสามารถตรวจสอบความคืบหน้าได้จากแถบ Progress Bar และกล่องข้อความ Log ด้านล่าง
6.  หากต้องการหยุดการทำงานกลางคัน สามารถคลิกปุ่ม **หยุดทำงาน** ได้ทันที

## 🔒 นโยบายความเป็นส่วนตัวและความปลอดภัย (Privacy & Security)

*   เครื่องมือนี้ทำงานในรูปแบบ Content Script ภายในเบราว์เซอร์ของผู้ใช้เท่านั้น
*   **ไม่มีการเก็บข้อมูล คุกกี้ เซสชัน หรือเนื้อหาเอกสารใด ๆ** และไม่มีการเชื่อมต่อหรือส่งต่อข้อมูลไปยังเซิร์ฟเวอร์ภายนอก
*   ระบบประมวลผลข้อมูลในหน่วยความจำชั่วคราวและล้างข้อมูลออกทันทีเมื่อปิดหน้าต่างการทำงาน
